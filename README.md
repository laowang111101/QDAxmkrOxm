# 医院病历管理系统 java393

## 项目概述

医院病历管理系统是一款基于B/S结构的Web系统，采用前后端分离的技术架构。系统使用ASP.NET作为后端框架，Vue、CSS、Html5和JS构建交互式前端界面，并以MySQL作为数据存储。整体采用MVC（Model-View-Controller）模式设计，确保了系统结构的清晰和易于维护。

## 技术栈

- **开发软件**：Visual Studio 2017及以上版本
- **开发语言**：.NET
- **后台框架**：ASP
- **数据库**：MySQL
- **前台技术**：Vue、CSS、Html5、JavaScript
- **浏览器**：兼容谷歌浏览器

## 功能模块

### 系统角色

- **管理员**：负责整个系统的后台管理。
- **用户**：可以进行预约挂号、管理个人病历等。
- **医生**：管理预约挂号，以及病历信息等。

### 功能列表

- **管理员模块**
  - 个人中心
  - 用户管理
  - 医生管理
  - 科室信息管理
  - 预约挂号管理
  - 取消挂号管理
  - 就诊评价管理
  - 病例信息管理
  - 预约时段管理
  - 留言板管理
  - 系统管理

- **用户模块**
  - 个人中心
  - 预约挂号管理
  - 取消挂号管理
  - 就诊评价管理
  - 病历信息管理

- **医生模块**
  - 个人中心
  - 预约挂号管理
  - 取消挂号管理
  - 就诊评价管理
  - 病历信息管理
